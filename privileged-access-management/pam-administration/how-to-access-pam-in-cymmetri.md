# How to Access PAM in Cymmetri

1. Navigate to your Cymmetri Tenant for e.g. sunstarcafe.newqa.cymmetri.in/login&#x20;
2.  Login into your tenant by entering your username and password of the Organization Administrator and Click "Login"

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027556188/original/0xAw6pJQ-6Qv2vGWhYdwI9svSSER1gCTVw.png?1677660905" alt=""><figcaption></figcaption></figure>
3.  Enter Cymmetri Authenticator Code and Click "Verify"

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027556225/original/397sdcCJ7KAlvOEfaiOi4RFaV_wgAuVTjw.png?1677660919" alt=""><figcaption></figcaption></figure>
4. Go To "Configurations"



<figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027557041/original/S9gqTuQFoVyhU5D2MMy83TfFI5d6qHUzvg.png?1677661324" alt=""><figcaption></figcaption></figure>

And Click on "Admins"

<figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027556985/original/a70DJoKKSVsSXetTYsKLJ9A-RiPamL-byA.png?1677661278" alt=""><figcaption></figcaption></figure>

5. Select **PAM Write Access** option and Click "**Add New**" to add a new PAM Write Access Administrator

<figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027557385/original/fxT7H12aRGwk-pwbcGIJrwnzq36ZvWCo-A.png?1677661581" alt=""><figcaption></figcaption></figure>

6.  Find the user who needs to be the Administrator and Click "Assign"

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027970775/original/GDD0gKTA182XhuqrSS-47WGh8YaOe6kn1Q.png?1678101798" alt=""><figcaption></figcaption></figure>
7.  Ensure correct Role is selected and Click "Save"

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027970951/original/8IMHLtNcQwRMtZ_tRwe98RLYvOtOEfRy_g.png?1678101846" alt=""><figcaption></figcaption></figure>
8.  Ensure the selected user is appearing in the list of Admins

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027971383/original/ddpySqqZDqBKMtZiF0gh2qMdxXdk7Q5T4Q.png?1678101922" alt=""><figcaption></figcaption></figure>
9.  Follow the same steps to assign a **PAM Read Access Administrator**

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027971814/original/QMJD5U5iXvq3Fh9ojV03mPHdX0YecImfaQ.png?1678102024" alt=""><figcaption></figcaption></figure>
10. And **PAM Report Access Administrator**

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027971940/original/6TTqA4jN-CLlPYkbepJTi1D2nq1OjvkzIw.png?1678102072" alt=""><figcaption></figcaption></figure>
11. Logout from the current login

<figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027972060/original/7daG-b0UKJx5xLqwUztzHxu_TtJ-QMEAuQ.png?1678102124" alt=""><figcaption></figcaption></figure>

12. And login again using the login credentials of the **PAM Admin** that was just added&#x20;

<figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027972281/original/MA66t6QR9oZLrAwB96VULQdLGeIjpkJaUQ.png?1678102190" alt=""><figcaption></figcaption></figure>

13. Since the user is now an administrator we will need to provide the authenticator code and click on "Verify" to login

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027558772/original/zk0N3Ggx5deP_kXapnEejF9Kbvk9KgwosQ.png?1677662354" alt=""><figcaption></figcaption></figure>
14. To Access PAM we need to Click on "Products"

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027558965/original/KPdZt15pE1kDhmEfnP874vb-AdLD7eP_WQ.png?1677662439" alt=""><figcaption></figcaption></figure>
15. And then select Privilege Access Management

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027559462/original/PkqF1RLpk3gSZelPhskwSwzWQirTYQlZIg.png?1677662757" alt=""><figcaption></figcaption></figure>
16. This opens the screen for Privilege Access Management as shown below

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027972918/original/nwawHzzBN_zLce46QBv2HT3VvFdJAvIimA.png?1678102379" alt=""><figcaption></figcaption></figure>
