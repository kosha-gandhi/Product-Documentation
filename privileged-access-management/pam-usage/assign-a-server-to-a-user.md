# Assign a server to a user

Once the PAM Server is configured by the PAM Administrator it can be assigned to users. Following are steps to assign a server to a user.

1. Click on Products

<figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027765173/original/FBKhFAWJW3n6dqr2qq71-vFXl3M7oqnpYg.png?1677842320" alt=""><figcaption></figcaption></figure>

2.  Then click on Privilege Access Management

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027765299/original/Ls1eGNvN0QH7DoriaGKOHkriAd92n7dXoA.png?1677842396" alt=""><figcaption></figcaption></figure>
3. Then click on devices

<div align="left"><figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027765489/original/IVuxxTgI3P9g7IouOAsh0jMFrp9Tk3aaSA.png?1677842478" alt="" width="188"><figcaption></figcaption></figure></div>

4.  Then click on the server that you want to assign to the user

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027765835/original/-FwywUKEmX4Qk4yw8zwhLwLOpAg6PQerOQ.png?1677842613" alt=""><figcaption></figcaption></figure>
5.  Then select **Assignments** and click on **Assign New** button

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027766216/original/uH1esrsy2vsvDWMI9l9SnoR0KowHgOwp7Q.png?1677842796" alt=""><figcaption></figcaption></figure>
6.  Choose the user to be assigned and click on **Assign**

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84028182055/original/e3Hhou0za6MxBWJNS6NopM6GvEoluFBSXg.png?1678273270" alt=""><figcaption></figcaption></figure>
7.  It then prompts you for the duration to provide Time Based Access. Select the desired duration and click on Save

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84027766830/original/Xbe1FPZIX702ZmSHKjTCazbR58NgTM9WYA.png?1677843076" alt=""><figcaption></figcaption></figure>
8.  It then shows that the user has been assigned

    <figure><img src="https://s3-ap-south-1.amazonaws.com/ind-cdn.freshdesk.com/data/helpdesk/attachments/production/84028182126/original/50cVef1WDO13fGD9kwUnKbmdqHYWX6bZiA.png?1678273333" alt=""><figcaption></figcaption></figure>
9. As soon as the assigned Date and Time occurs the assigned user can be seen in the list of assigned users.
